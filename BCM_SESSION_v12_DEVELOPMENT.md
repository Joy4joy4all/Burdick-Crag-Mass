## THE LAMBDA MODULATOR — CORE CONCEPT

### What Lambda Is (from the solver)

Lambda (λ) is the decay rate of the substrate memory field.
In the BCM solver: λ = 0.1 (global default).

Physical meaning:
- High λ → fast decay → substrate forgets quickly →
  weak gravitational memory → shallow well
- Low λ → slow decay → substrate remembers longer →
  strong gravitational memory → deep well

Lambda determines how quickly the substrate releases its
memory of agitation at a given location.

### The Gradient Principle

Every massive body creates a sigma (σ) memory well in the
substrate. The gradient between wells defines a natural
flow direction — substrate flows from high-σ to low-σ
regions. This gradient IS gravity in the BCM framework.

A ship sitting in the Sun's sigma well experiences the
gradient as gravitational pull toward the Sun. To move
outward (toward Saturn), the ship must either:

A) Fight the gradient (conventional thrust — expensive)
B) Modify its local lambda to reshape the gradient
   (substrate surfing — efficient)

### The Modulator

A lambda modulator is a hypothetical device that locally
adjusts the ship's coupling to the substrate:

- INCREASE λ behind the ship → substrate forgets faster
  behind → local sigma drops → shallow well forms aft
- DECREASE λ ahead of the ship → substrate remembers
  longer ahead → local sigma rises → deep well forms fore

The net effect: an asymmetric sigma gradient centered on
the ship. The substrate's own maintenance flow carries the
ship forward along the steepened gradient.

No fuel consumed. No exhaust produced. No momentum
exchanged with a reaction mass. The ship borrows coherence
from the local substrate budget and returns it as it
passes through. Net zero maintenance cost for transit.
